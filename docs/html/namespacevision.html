Detailed Description

Starts a ROS service that takes a sensor_msgs/Image and a _____ as input, elaborates the data in order to detect and locate the blocks on the working table, and returns geometry_msgs/Pose[] as output.

The blocks are detected via a YoloV8 model, from which a prediction (and its confidence) is obtained. The point cloud is used to check and verify the prediction of the model, as well as to locate the block on the table with (x, y, z) coordinates with respect to the world frame.

The blocks that are returned in geometry_msgs/Pose[] as output are not always all the detected ones, but only the ones with a high probability of being correctly identified. This is why the service may need to be run multiple times until the boolean geometry_msgs/Pose[]/finished is returned as True.
